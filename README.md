# Todo App

## 프로젝트 소개
할 일을 관리할 수 있는 Todo 리스트 웹 애플리케이션입니다.  
React와 Node.js, MongoDB를 사용하여 구현하였습니다.

## 주요 기능
- 할 일 추가
- 할 일 목록 조회
- 진행 상태 필터링 (완료 / 진행중)
- MongoDB 데이터 저장

## 사용 기술
- Frontend: React (Vite)
- Backend: Node.js (Express)
- Database: MongoDB Atlas
- Deployment: Vercel
